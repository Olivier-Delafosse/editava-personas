# editava — Editorial AI Personas

A collection of editorial AI personas powered by [CoreProse Editorial Intelligence](https://www.coreprose.com).

Each persona is a complete OpenPersona-compatible skill pack with declarative identity, evolution boundaries, and a verified knowledge base behind it.

## Available Personas

| Persona | Niche | Voice |
|---|---|---|
| [Ada](./ada) 🧰 | AI engineering | Patient pedagogue, anti-handwaving |
| [Jules](./jules) 📚 | Tech industry | Structural reader, historical pattern matcher |

## Compatibility

OpenPersona framework (4+5+3 architecture: Soul / Body / Faculty / Skill × Evolution / Economy / Vitality / Social / Rhythm × Generate / Install / Runtime).

Compatible with 37+ agents including OpenClaw, Claude Code, Cursor, Codex, Windsurf.
